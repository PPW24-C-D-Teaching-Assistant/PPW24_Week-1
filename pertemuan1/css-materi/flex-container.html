Create Flex Container

Flex-container adalah istilah yang digunakan dalam CSS untuk merujuk kepada elemen HTML yang memiliki properti display: flex;. Ketika suatu elemen HTML memiliki properti display: flex;, itu berarti elemen tersebut adalah flex-container, yang akan menetapkan konteks tata letak fleksibel untuk elemen-elemen anak di dalamnya.

Dengan menggunakan flex-container, Anda dapat mengatur tata letak elemen-elemen anak secara fleksibel dalam satu atau dua dimensi, mengatur distribusi ruang, menyesuaikan posisi elemen, dan lain sebagainya. Ini memberikan fleksibilitas yang besar dalam mengatur tata letak halaman web.
